# Library Management System 📚

A simple Library Management System built using C# and Object-Oriented Programming (OOP) principles.  
The project manages different types of library items, users, and borrowing records in a clean and structured way.

---

## ✨ Features
- Add new library items (Book, Magazine, DVD)
- Register new users
- Borrow and return items
- Track borrowing history
- Apply core OOP concepts:
  - Inheritance
  - Polymorphism
  - Encapsulation
  - Abstraction
